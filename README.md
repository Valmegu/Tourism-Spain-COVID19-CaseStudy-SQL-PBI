## Descripción

Este proyecto personal para practicar habilidades de Análisis de Datos y storytelling. 
Gracias a los datos publicados por el Instituto Nacional de Estadística podremos ver claramente como la pandemia del COVID en 2020 y confinamiento afectaron el turismo en España.
El Objetivo es *analizar que tan fuerte fue el descenso de turistas* registrado en el país y *determinar si el país ya se ha recuperado*
